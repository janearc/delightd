# Contributing

This repository uses the [vouch](https://github.com/mitchellh/vouch) trust
protocol. `VOUCHED.td` at the repository root lists the handles trusted to
participate here; a denounced handle (leading `-`) is blocked.

Being vouched grants **participation, not write access**: vouched handles are
trusted to open issues and pull requests. Landing and merge remain the
maintainer's decision — vouch confers trust to contribute, never the commit bit.
